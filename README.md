### Java-mid1
* Object class
  * 공통 기능을 가지기 위해 최상위 클래스로 Object 클래스를 상속 받는다.
    * toString
      * 객체(인스턴스) 의 정보를 제공 
    * equals
      * 객체의 같음을 비교
      * 동일성 ==
        * 같은 객체 (주소값) 인지 확인
      * 동등성 equals
        * 두 객체가 논리적으로 동등한지 확인 
    * getClass
      * 클래스의 정보를 제공
  * 다형성의 기본구현
    * 최상위의 클래스 이므로 모든 클래스를 담을 수 있다.
